# 🤖 AI-Powered Document Q&A Chatbot with Bulk Email Automation

An intelligent AI application that enables users to upload PDF and DOCX documents, ask natural language questions about their content, and automate bulk email communication using AI.

Built using Retrieval-Augmented Generation (RAG), FastAPI, Gemini AI, ChromaDB, and Resend Email API.

---

## 🚀 Features

- 📄 Upload PDF and DOCX documents
- 🤖 Ask questions in natural language about uploaded documents
- 🔍 Semantic search using ChromaDB vector database
- 🧠 Retrieval-Augmented Generation (RAG)
- 📑 Multi-document support
- 📧 Extract email addresses automatically from uploaded files
- ✉️ Generate professional emails using Gemini AI
- 📨 Send bulk emails to extracted recipients using Resend API
- 🎤 Voice input support
- 🗑️ Upload, manage, and delete documents
- ⚡ Responsive web interface

## ⚙️ How It Works

1. Upload one or more PDF/DOCX documents.
2. Documents are converted into text.
3. Text is split into chunks.
4. Sentence Transformers generate embeddings.
5. Embeddings are stored in ChromaDB.
6. User asks a question.
7. Relevant document chunks are retrieved.
8. Gemini AI generates an accurate answer based on the retrieved context.
9. Email addresses can be extracted automatically from uploaded files.
10. AI generates professional email content.
11. Emails are sent in bulk using the Resend API.
